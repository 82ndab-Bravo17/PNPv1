 PLATINUM NUKE Pro v2 INSTALLATION GUIDE

Automatic installation:

 1. Go into your hosting control panel - Use MYSQL or phpmyadmin, depending on how your server is setup and create a database.

 2. Create a database user and password and assign it to the database you just created. Remember that information.

 3. Upload everything inside the 'public_html' folder to your hosting space where you would like to install Platinum Nuke Pro.

 4. The installer is designed to run automatically once you go to your site URL like: http://www.yourdomain.com

 5. Answer all the questions the installer asks.  DO NOT RENAME THE ADMIN FILE AT THIS TIME.  Leave it as admin.php for now.  You can rename it once the installation is done if you want. (Adds a little more security if you rename it.)   If you choose to, rename the admin.php file then change the $admin_file setting to the new name in the config.php file.  You must also change the admin.php file name in the robots.txt file to the new name.

 6. Once the installer has completed your must then delete the install folder. (The installer should automatically remove this folder, but double check anyways.) Then you can log in to your site as user and/or admin and start customizing.

 7. If for some reason the automatic installer does not work for you, you will need to do a manual install:

Manual installation:

 1m. Enter the new database information (from steps 1 & 2 above) into the config.php file that is now on your server. (It will be located in the site's root folder.) in the section that looks like this:

    $dbhost = "localhost";
    $dbuname = "database_user_name";
    $dbpass = "database_user_pass";
    $dbname = "database_name";

Once you have edited the config.php file and uploaded to your hosting space, go to your phpMyAdmin or whatever you use to do database work and import the file 'PNPv1_MAIN.sql' into the database you made earlier. (This will install the database tables Platinum Nuke Pro requires to run.)

 2m. Go to your site url (http://www.yourdomain.com) and follow the instructions in the middle of the page under the Platinum Nuke Pro logo.

 3m. IMPORTANT: Make sure the install folder has been deleted from your hosting space.
